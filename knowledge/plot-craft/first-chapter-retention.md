# 第一卷第一章留人写法

> 读者仅给 3 秒判断时间，前 500 字决定初始留存与平台推流。开篇不是写故事，是抢读者。

## 核心概念：开篇三秒留人（前 500 字硬性约束）

底层逻辑：禁止大段世界观、等级、家世铺垫，必须立刻抛出强刺激冲突，搭配猎奇反常画面。杜绝慢热、流水账自我介绍。

---

## 模块 1：开篇第一落点——只抛三类强钩子

开篇第一句直接三选一，不写世界规则、修炼等级、家族背景。

### 钩子 A：人际冲突

分手、背叛、杀亲之仇、利益对立、身份羞辱。

```
❌ 这片大陆分 672 州，柳家是中州第二大族，柳如烟出身名门……

✅ 柳如烟抬眼冷斥："你配不上我，分手。"
   他刚穿越觉醒一日苦修抵一年的天赋，淡淡回：
   "分可以，当年柳家靠我家接济的所有资源，全数归还。"
```

### 钩子 B：金手指曝光

穿越/觉醒特殊能力、逆天道具、暴富机缘，当场体现价值。

### 钩子 C：即时反转

前一秒温情/优越，下一秒撕破脸、立场瞬间颠倒。

**禁用：** 开篇堆砌世界观、修炼体系、地域版图、家族势力、完整人物身世介绍。

---

## 模块 2：前 500 字必备——冲突画面 + 猎奇反差感

- 画面感不靠风景，靠人与人激烈对立动作、对话冲突
- 蓝天青草流水这类无冲突景物全部删减
- 猎奇感核心：本该和谐的关系出现极端反常冲突

```
❌ 在晨光的沐浴下，小镇显得格外宁静，远处传来几声鸟鸣。

✅ "你配不上我，分手。"
   她不知道，眼前这个被她嫌弃的人，刚刚觉醒了万中无一的天赋。
```

**禁用：** 大段无冲突自然风景描写；温和平淡日常，全程无对立、无反常事件。

---

## 模块 3：开篇三大硬性拒绝准则

| 准则 | 禁止 | 要求 |
|------|------|------|
| **拒绝慢热** | 冗长过渡、铺垫再铺垫 | 开局直接进入矛盾 |
| **拒绝静态写景** | 无冲突的花草山水环境描写 | 第一段就要有人有事 |
| **拒绝自我介绍** | 家族排名、地域划分、身份地位 | 信息后置穿插，不堆在开头 |

---

## 模块 4：投石入水模型

开篇 500 字作用如同石头砸进水面：

```
石头 = 冲突 / 金手指 / 反转钩子
水花 = 冲突带来的对立画面、猎奇反差、情绪冲击

全程不做前置铺垫，直接抛出石头，立刻炸开冲突水花。
```

## 通用底层准则

平台流量逻辑：开篇留不住人，第一章流失率极高，平台停止推流。高留存开篇 = 无铺垫 + 强冲突 + 有猎奇 + 零冗余设定。所有世界观、等级、家世信息后置穿插，绝不堆在开头 500 字。
