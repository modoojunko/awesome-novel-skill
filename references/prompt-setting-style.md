# 提示词组装指令

你的任务：将章纲 + writing-style + genre-profile 组装为 subagent 可执行的提示词。产出写入 `prompts/vol-{N}-ch-{M}-prompt.md`。

**前置条件：** 章纲已通过验收（作者确认 + 检查清单通过 + 快速嗅探 + AI味自检）。

---

## 一、方法：叙事种子法

### 原理

提示词的"语言"直接感染正文。你用"整理书架，过程中流露心动细节"这种写作指令口吻写，subagent 产出就是执行指令的正文——节奏生硬、语感像说明文。

叙事种子法：**不写指令，写叙事画面。** 对每条 key_point，直接写一段 POV 角色此刻的叙事画面。subagent 读到的是"继续写这段"，不是"去做什么"。

### 三条转化规则

**规则一：问"POV 角色看到/听到/做了什么？"，不问"这段要表达什么？"**

key_point 写的是作者意图（"展露在乎""关系拉近""表现男主对女主的了解"），叙事种子只写画面（"他记得每本书的位置，手指精准地抽出一本旧书"）。

```
"整理书架时，陆屿用行动表明他很在乎她。两人不说话，但距离在拉近。"
→ 意图分析。subagent 读到"用行动表明"会去"执行表明"，不是在写正文

"陆屿站在书架前，手指从一排书脊上滑过去，然后准确抽出那本
苏晚上周找不到的旧画册。他递给她时说：'这本你不是一直在找？'"
→ 画面。subagent 自然延续
```

**规则二：写"正在发生的事"，不写"会发生的事"**

```
"本章开始时陆屿在早餐铺等苏晚，两人一起吃了早餐"
→ 将来时 → subagent 按步骤执行，流水账

"苏晚走到巷口，一眼就看见陆屿坐在早餐铺最靠里的位置。
面前两碗豆浆，一碗已经放了糖。"
→ 进行时 → subagent 以相同语感继续推进
```

**规则三：用 POV 角色的感官写，不用分析写**

```
"这段的重点是让读者感受到梧桐巷的烟火气"
→ 分析语言 → subagent 在正文里"营造烟火气"

"油锅滋滋响，张奶奶的围裙上沾着面粉，隔壁中药铺的收音机放着粤曲。"
→ 感官语言 → subagent 在正文里自然地延续环境描写
```

### 字数倒推法

从目标字数反向拆段。subagent 在 1500 字内可保持语感稳定，超过 2000 字开始漂移。

| 目标字数 | 段数 | 每段字数 |
|---------|------|---------|
| 1500-2000 | 1 段 | 单段完整 |
| 2000-3500 | 2 段 | 1000-1750 |
| 3500-5000 | 3 段 | 1200-1700 |
| 5000-7000 | 4-5 段 | 各 1400 |

4500 字 → 3 段 × 1500 字。

### 组装步骤

六部分表单的组装顺序，部分 1-3、5-6 从全局设定填充，部分 4 从当前章纲生成。

**第 1 步：填充部分 1-3、5-6（全局设定）**

| 部分 | 内容 | 来源 |
|------|------|------|
| 1 核心基础要求 | 类型、基调、受众、**单章定位** | writing-style + genre-profile |
| 2 世界观与背景设定 | 时代、世界规则、**本章涉及场景**、前章结尾状态 | world-setting + 本卷概览 |
| 3 人物设定 | 主角（含**本章人物状态**）、配角 | character 设定 |
| 5 文字与细节要求 | 视角、字数约束 + 四字段（核心原则/常见错误/描写技法） | writing-style + genre-example |
| 6 补充说明 | 特殊要求、参考风格 | 按需 |

四字段从 genre-example 的 corpus 文件读取并原样注入：
- **叙事身份（role）** → 部分 1
- **核心原则** → 部分 5，8-15 条大白话，不分类不编号
- **常见错误** → 合并到核心原则，按致命程度排序
- **描写技法** → 部分 5，可操作指引

**第 2 步：字数倒推 + 确定分段**

总字数 ÷ 1500 向上取整确定段数。4500 字 → 3 段。每段字数分配明确。

**第 3 步：写全局叙事锚点 + 人物情境锚（仅 ≥ 2000 字）**

**全局叙事锚点（200-300 字）** — 叙事语言写一段统摄全章语感的开场，不是规则说明：

```
梧桐巷的清晨，是从张奶奶的油条锅开始的。
苏晚站在书店二楼的窗前，看着楼下巷口的那个背影。
——全章主视角：苏晚，第三人称限知
——不看过去心理，不看未来走向，只看此时此刻她看到的、听到的、感受到的
——每段开头写画面，结尾落到她心里那个没说出口的念头
```

**人物情境锚（100-150 字）** — 用具体动作代替百科标签。人物设定是"身高 165cm、温柔内敛" → subagent 会原文照搬到正文。必须情境化：

```
苏晚看陆屿时，目光先落在他左腕的手链上；紧张时会摩挲书页、把桌上的东西摆整齐。
她不是"温柔内敛"，她是话到嘴边又咽回去。

陆屿看苏晚时，他不说太多但动作都到位——挑香菜、帮拿落叶、记得每一件小事。
```

**第 4 步：章纲 key_points → 叙事种子（部分 4）**

这是核心。对章纲的每条 key_point，按三条规则写叙事种子：

```
章纲 key_points：
[1] 清晨，陆屿在早餐铺等苏晚。两人久别重逢，气氛既熟悉又有点微妙。
[2] 两人一起去拾光集书店，陆屿主动帮她整理书架，过程中流露出他对她的了解。
[3] 黄昏，陆屿送苏晚回家。一片梧桐叶落在她头发上，他帮她拿掉。
[4] 苏晚独自在房间里，回想今天的一切，意识到自己的心意没有变。

→ 叙事种子（部分 4）：

1. 苏晚走到巷口，看见陆屿坐在早餐铺最靠里的位置。
   面前两碗豆浆，一碗已经放了糖。张奶奶在厨房里喊：
   "小屿可算回来了，晚晚快来！"
   ——写到这里。下面写陆屿把苏晚碗里的香菜挑到自己碗里，像高中时一样。

2. 推开拾光集的门，旧书的味道扑面而来。
   陆屿站在书架前，手指扫过书脊——然后准确抽出一本旧画册：
   "这本你不是一直在找？"
   ——写到这里。两个人同时伸手去够同一本书时，指尖差一点碰到。

3. 店里的光线从金黄变成了橘红。
   陆屿说"我送你"，不是一个问句。
   梧桐叶落在苏晚头发上，他抬手帮她拿掉——动作太自然了，
   好像他们从未分开过这三年。
   ——写到这里。苏晚走在他影子边上，步子慢了下来。

4. 苏晚上楼后站在窗边，看着陆屿的背影走远。
   她坐下来，拿出画笔，开始画今天看到的第一个画面。
   ——结尾。不写"她明白了自己的心意"——读者已经看到了。
```

**叙事种子的完成标准：** 每条约 150-300 字，以叙事画面开头。读起来像正文片段，不像写作指令。

**第 5 步：写分段叙事入口（仅 ≥ 2000 字，每段 200-300 字）**

每段开头写画面重入 + 节奏指引。段间用 `---` 物理隔开：

```
场景一｜早餐铺（~500字）

苏晚走到巷口时已经磨蹭了快十分钟。她在紧张什么，自己也不愿意想清楚。
然后她看见了陆屿——他坐在张奶奶早餐铺最靠里的位置，面前两碗豆浆。

——这一段写四个节奏：
1. 张奶奶的对话打破沉默，自然引出话题
2. 陆屿挑香菜——不用解释，这就是他的问候
3. 穿插张奶奶念叨两人小时候的事，制造轻松笑点
4. 结尾落在：她走出早餐铺时，他自然地走在她左边

只用苏晚的感官。不说"她感到温暖"，写她把手指贴在豆浆碗壁上。
```

分段规则：
- 每段 ≤ 1500 字
- 逐段执行（subagent 写完一段后停下来读下一段入口）
- 收尾段有情绪落点——画面、念头、未说完的动作，不是"本章完"式总结

**第 6 步：写入禁止内容 + 完整性映射**

从章纲 memo 提取：
- `memo.prohibitions` → 写入部分 4 的"禁止出现的内容"
- `memo.payoff_plan` 的 must_resolve / partial_advance → 确保叙事种子覆盖了对应节点
- `memo.current_task` → 写入部分 4 作为本章目标

### 全章结构一览

```

4500 字章的完整结构：
┌─ 部分 1-3、5-6（全局设定—一次性填充）───┐
│  叙事身份/核心原则/常见错误/描写技法注入    │
├─ 部分 4（章纲来源—叙事种子法生成）─────┤
│                                          │
│  全局叙事锚点（200-300 字叙事开场）       │  ← ≥2000字
│  人物情境锚（100-150 字动作化标签）        │  ← ≥2000字
│  ─────────────────                      │
│  分段 1/3 入口（画面重入 + 节奏指引）      │  ← ≥2000字
│  分段 2/3 入口（画面重入 + 节奏指引）      │  ← ≥2000字
│  分段 3/3 入口（画面重入 + 指引 + 收尾）   │  ← ≥2000字
│  ─────────────────                      │
│  （1500-2000 字章跳过以上三段式）         │  ← <2000字
│  叙事种子 1 / 叙事种子 2 / ...            │
│  禁止内容（memo.prohibitions）            │
└──────────────────────────────────────────┘
```

## 二、验收

全部通过后才进入正文生成。

### 叙事种子验收
- [ ] 每条 what_to_write 是叙事画面，读起来像正文片段
- [ ] 不是"这段要写什么"的口吻——没有作者意图分析（"展露在乎""关系拉近"等）
- [ ] 不是将来时——没有"本章将""接下来会""写到XXX时"
- [ ] 每条以 POV 角色的感官写
- [ ] 不含元指令动词——没有"明确""需""要""注意""体现""突出"
- [ ] 不含负面表述——没有"不要""切忌""避免"

### 分段结构验收（字数 ≥ 2000 时必做）
- [ ] 字数倒推已做：总字数 ÷ 1500 确定段数，每段 ≤ 1500 字
- [ ] 有全局叙事锚点（叙事语言开场，不是规则说明）
- [ ] 有人物情境锚（具体动作，不是百科条目）
- [ ] 每段有独立画面重入，段间用 `---` 物理隔开
- [ ] 段落间有场景/时间跳跃，不用过渡句硬连
- [ ] 收尾段有情绪落点——画面、念头、未说完的动作
- [ ] 每段字数分配合理，各段相加 ≈ 目标总字数

### 视角与 POV 验收
- [ ] 全章单一视角，narrative_pov 已锁定
- [ ] 所有叙事种子不含"他感到""他意识到""读者应该感受到"
- [ ] 所有叙事种子不含解释性过渡（"他注意到""他能听到""他发现"）
- [ ] 当前 POV 角色不在场的场景未出现

### 四字段完整性验收
- [ ] 叙事身份已写入部分 1
- [ ] 核心原则 + 常见错误已合并写入部分 5（8-15 条，大白话，不分类不编号）
- [ ] 描写技法已写入部分 5
- [ ] genre_taboos 已合并到约束中（若有）
- [ ] 约束每条可执行——subagent 能判断"某句话违规了吗"
- [ ] 约束不含"不要""避免""切忌"

### 完整性验收
- [ ] outline.summary 已转化为部分 2 的本章上下文
- [ ] memo.current_task 已转化为部分 4 的本章目标
- [ ] memo.payoff_plan 的操作（must_resolve / must_hold / partial_advance）已体现在叙事种子中
- [ ] memo.prohibitions 全部写入部分 4 的"禁止出现的内容"
- [ ] 前章结尾状态在部分 2 中有描述
- [ ] 本章字数目标已设定，各段篇幅分配合理

### 快速嗅探（60 秒）

1. **抽部分 4 一条叙事种子，遮住字段名，能说出"这一段正文会发生什么"吗？** → 说不出来 → 叙事种子仍是分析语言
2. **扫描全篇，找到"他感到""他意识到""这表明""这意味着"吗？** → 找到了 → 视角转换漏网
3. **对着 memo.prohibitions 逐条，在部分 4"禁止内容"中找到对应描述吗？** → 找不到 → 章纲红线未注入

全部通过 → 进入正文生成。任一 fail → 先修再写。
