# 角色设定参考

> 逐层聊四层七字段，聊完写进 `settings/character-setting/<id>.yaml`。
> 不解释概念，直接聊。每聊完一层，过本节自检再进下一层。

---

## 原则 1：角色是决策引擎，不是履历表

**规则：** 七字段填完要能推断角色在任何情境下会怎么想、怎么做。填了一堆信息但推不出行为 → 那是履历表，不是角色设定。

✅ 正确：
> cognition: 习惯从利益角度分析一切——任何人做事都有动机，找不到动机就说明信息不够。不信任"善意"这种解释。
> → 遇到有人无偿帮忙，角色会先怀疑对方有什么企图。

❌ 错误：
> 性格：聪明、冷静、理性
> → 标签堆砌。"聪明"怎么表现？冷静到什么程度？遇到亲友遇险还冷静吗？

**判断标准：** 给出意外情境（"有人当街刺杀他旁边的陌生人"），能否不假思索说出角色会怎么做？说不出来 → 设定不够具体，继续聊。

---

## 原则 2：主角必须有真缺点

**规则：** "太正直""太善良""太信任别人"不是缺点——它们是换种说法夸角色。真缺点是让角色在关键时刻**做错选择**的东西。

✅ 正确：
> 冲动（遇到亲友遇险直接冲上去，不计后果）
> 多疑（谁都信不过，需要帮助时张不开口）
> 自卑（觉得自己不配被救，关键时刻放弃求救）

❌ 错误：
> "他太善良了" / "他太正直了" / "他太信任别人了"
> → 这些是优点的变体。读者能感觉到作者在夸角色。

**判断标准：** 这个缺点是否会在第 3-5 章让角色做出错误决定？不会 → 这不是真缺点，换个。

---

## 原则 3：配角有自驱动目的

**规则：** 配角不是为主角存在的。拿走主角，配角有自己的欲望和恐惧，能独立行动。

✅ 正确：
> 林燕是暗网线人，她帮主角是因为欠人情，但她有自己的生存法则——不站队、不出头、活得久。必要时会为自保而背叛主角。
> → 拿走主角，林燕还是那个在灰色地带求生的女人，继续做她的情报买卖。

❌ 错误：
> 老张是主角的朋友，总是在主角需要的时候出现帮忙。
> → 工具人。拿走主角，老张就消失了。

**判断标准：** 把主角从故事里拿走，这个配角还有事做吗？没有 → 退回加自驱动目的。

---

## 怎么聊：四层七字段逐层聊

从内到外，按顺序聊。每层依赖内层已经定好，不跳层。

### 第一层：核心层（驱动力）

聊清楚"角色为什么这么做事"。核心层定了，角色遇到任何情境的行为基线就定了。

按顺序问：

1. **cognition："你角色怎么看人和事？有什么固定的判断模式？"**
   → 不是"聪明/笨"，是过滤信息的框架。"老江湖看谁都像骗子"和"书生相信人性本善"遇到同一件事反应完全不同。
2. **worldview："角色相信什么、不相信什么？有什么事是ta认定的道理？"**
   → 信念是用来冲突的——主角和反派的 worldview 针锋相对时，冲突不需要额外设计。
3. **self_identity："角色觉得自己是什么样的人？——注意不一定是他实际的样子。"**
   → 认知偏差就是戏剧空间。角色自认"冷静理性"但实际冲动，他会在自己都没意识到的瞬间冲出去。

**设计技巧：** 找核心层的矛盾点。self_identity 偏差（自认冷静实际冲动）、worldview 和 values 的张力（相信强者制定规则但又认为不能欺压弱者）——这种内在矛盾让角色有层次。

✅ 正确：
> cognition: 从利益角度分析一切，不信任"善意"。
> worldview: 强者制定规则，弱者遵守规则。抱怨没用，变强才有用。
> self_identity: 觉得自己冷静理性谋定后动，实际遇到亲友遇险第一反应是冲上去。

❌ 错误：
> 性格：聪明、勇敢、正直
> → 全是标签，没有认知框架、没有信念、没有自我认知偏差。

**本节自检（过完再进准则层）：**
- [ ] cognition 是认知框架（怎么看世界），不是性格标签（聪明/勇敢）？
- [ ] worldview 是具体的信念（可与其他角色冲突），不是抽象概念（相信正义）？
- [ ] self_identity 有认知偏差（角色自认为的和实际有差距）？

---

### 第二层：准则层（边界）

聊清楚"什么事角色打死不做、什么事会破例"。价值观是用来被挑战的。

问这一句就够了：
> "有什么事是角色打死不会做的？——如果非做不可呢？什么条件会让他破例？"

✅ 正确：
> values: 不背叛兄弟。不欺压平民。可以利用对手但不能侮辱对手的尊严。
> → 每个都能被挑战——如果背叛兄弟能救更多人呢？

❌ 错误：
> values: 重视友情、善良正直
> → 太抽象。重视到什么程度？会为朋友杀人吗？

**判断标准：** 能否给每个 value 设计一个"如果……会怎样"的挑战场景？不能 → 太抽象，继续问细节。

**本节自检（过完再进能力层）：**
- [ ] values 至少 2 条具体的行为边界（不是抽象美德）？
- [ ] 每条 value 都能被"如果……会怎样"挑战？

---

### 第三层：能力层（工具）

聊清楚"角色能做什么、不能做什么"。能力必须有具体表现和上限，不是头衔。

按顺序问：

1. **abilities："角色实际能做什么？输出上限在哪？"**
   → "金丹境"不是能力——"金丹境能御剑飞行"才是。能打几个？笼罩范围多大？
2. **skills："学过什么技术？从哪学的？有什么局限？"**
   → 技能有来源就有局限。码头帮工学的市井术在朝堂上没用。

✅ 正确：
> abilities: 十步内察觉杀气；单手拉八十斤弓，三箭连发不脱靶；过目不忘但需要时间回忆。
> skills: 青帮底层混出来的市井生存术（听脚步辨人数、看衣着断身份）；跟账房先生学过珠算记账。

❌ 错误：
> abilities: 武功高强；skills: 精通琴棋书画
> → "高强"多高？"精通"什么程度？来源和局限呢？

**判断标准：** 每个 abilities 能不能答出"范围、上限、限制条件"？不能 → 不够具体。

**写进文件的内容：**
- abilities（至少 2 项，每项有具体表现和边界）
- skills（至少 1 项，有来源说明）

**本节自检（过完再进处境层）：**
- [ ] abilities 有明确输出上限（能打几个/笼罩范围多大）？
- [ ] skills 有来源交代（从哪学的/为什么会的）？

---

### 第四层：处境层（现状）

聊清楚"角色现在站在什么位置、有什么包袱"。出身决定行为基线——穷小子的选择和富公子不会一样。

问这两句：
1. "角色什么出身？这对ta现在有什么影响？"
2. "角色现在的处境是什么？有什么甩不掉的包袱？"

✅ 正确：
> environment: 苍梧山脚猎户之子，父母死于妖兽袭村，被青帮老掌柜收养。在码头帮工长大，见过三教九流。

❌ 错误：
> environment: 出身贫寒
> → 贫寒到什么程度？当前有什么具体包袱？这和故事主线有什么关系？

**判断标准：** 处境能不能解释角色在第一章为什么处在当前的位置？不能 → 处境和剧情脱节，继续聊。

**写进文件的内容（完整清单）：**
- 角色 id（拼音，英文文件名用）
- 角色 name（中文展示名）
- story_role（protagonist / antagonist / supporting）
- 四层七字段全部有值
- 每条设定能用行为验证（不写读心结论，写行为证据）

**本节自检（过完进输出）：**
- [ ] 出身和处境与世界设定一致（角色在世界观里有对应位置）？
- [ ] 当前处境能解释"主角为什么第一章在这个位置"？

## 常见错误

| 错误 | 表现 | 改法 |
|------|------|------|
| 标签代替设定 | "高冷/腹黑/傲娇" | 拆成 cognition/worldview——为什么高冷？什么经历导致的？ |
| 假缺点 | "太正直/太善良/太信任别人" | 换成让角色做错选择的真缺点（冲动/多疑/自卑） |
| 工具人配角 | 配角只为主角存在，拿走主角就消失 | 给配角自己的欲望和恐惧 |
| 能力无边界 | "武功高强""能力很强" | 加具体上限——能打几个、范围多大、什么情况会失效 |
| 处境和剧情脱节 | 出身写了一大堆，但和主线毫无关系 | 每个处境信息都要能回答"这对剧情有什么用" |
| 代入感错位 | 主角是读者代入对象，不是读者仰望对象 | 主角不是完美偶像，是让读者觉得"换我也可能这么做"的人 |

## 输出去哪

`settings/character-setting/<拼音id>.yaml`，每角色一文件。归档时追加 state_history 和 emotional_arc。

## 案例

以下案例供聊到对应角色类型时参考。

### 案例一：仙侠玄幻——主角沈牧

```yaml
id: shen-mu
name: 沈牧
story_role: protagonist

cognition: 习惯从利益角度分析一切——任何人做事都有动机，找不到动机就说明信息不够。不信任"善意"这种解释。

worldview: 强者制定规则，弱者遵守规则。想改变处境就要变强，抱怨没有用。

self_identity: 觉得自己是个冷静理性的谋定后动者——但实际遇到亲友遇险时第一反应是冲上去，理性是事后找补的。

values: 不背叛兄弟。不欺压平民。可以利用对手但不能侮辱对手的尊严。

abilities: 能在十步内察觉杀气来源；单手拉八十斤弓，三箭连发不脱靶；过目不忘，但需要时间回忆。

skills: 青帮底层混出来的市井生存术（听脚步辨人数、看衣着断身份）；跟账房先生学过珠算和记账。

environment: 苍梧山脚猎户之子，父母死于妖兽袭村，被青帮老掌柜收养。在码头帮工长大，见过三教九流。
```

### 案例二：都市异能——配角（暗网线人）

```yaml
id: lin-yan
name: 林燕
story_role: supporting

cognition: 从小到大靠察言观色活下来的——谁在说谎、谁有杀意、谁能惹谁不能惹，三句话内判断。从不当面拆穿，记在心里、用到的时候翻出来。

worldview: 这世道没有黑白，只有"惹不惹得起"。特管处和暗网都不是好东西，但跟他们作对的人更惨。

self_identity: 觉得自己是个小心翼翼的中间人，不站队、不出头、活得久。但偶尔会在不该出手的时候出手帮人——事后自己都无法解释。

values: 不碰毒品和人口买卖。可以出卖情报但不能出卖命。欠的人情一定要还。

abilities: 能记住见过一面的人的脸和声纹，三年后还能认出来。开锁——从普通挂锁到电子密码锁，都能在三分钟内打开。伪装——换了装束和走路姿势后，熟人迎面走过都认不出。

skills: 江城市区活地图——知道所有监控盲区、暗巷、废弃建筑。跟老锁匠学过三个月开锁，后来自己练出来的。

environment: 江城市棚户区长大，父亲坐牢、母亲在工厂打工。十三岁开始混街头，给暗网跑腿送过三年货，知道太多不该知道的事。特管处不知道她的存在。
```

## 两步总检（聊完后走）

全部聊完、写入文件后，走两步自检。不过不进下一阶段。

### 第一步：格式检查

- [ ] 文件名是拼音 id，不是中文？
- [ ] story_role 是三个之一（protagonist / antagonist / supporting）？
- [ ] 七字段全部有值，没有空字段？

### 第二步：可用性检查

| 检查项 | 怎么查 | 不通过处理 |
|--------|--------|-----------|
| 行为可推断 | 给角色意外情境，能说出角色反应？ | 说不出→退回，设定不够具体 |
| 主角真缺点 | 缺点是否会在第 3-5 章导致角色做错选择？ | 假缺点→退回重写 |
| 配角自驱动 | 拿走主角，配角还有事做吗？ | 工具人→退回加自驱动目的 |
| 能力有边界 | 每个 abilities 有明确输出上限？ | "很强"无边界→退回加限制 |
| 与世界一致 | 出身和处境在世界观里有对应位置？ | 脱节→退回调整 |
