# 题材选择指南

> 先在 genre-corpus/index.md 浏览 24 种预置题材，引导作者选定主类型。题材是 Phase 2 第一步——定了题材再聊世界观和角色。
> 选完后写入 `settings/genre-setting.md`。

---

## 原则 1：题材决定读者预期

**规则：** 题材定义的是"读者翻开第一章时在等什么"。不是标签，是预期管理。

✅ 正确：
> 选仙侠 → 读者等境界突破、秘境夺宝、打脸反杀。
> 选悬疑 → 读者等线索推进、反转、真相揭露。

❌ 错误：
> "我的故事什么题材都沾一点，不分那么细。"
> → 不分题材就没法管理读者预期。可以叠加变体，但得有主类型。

**判断标准：** 选完后，能不能说清"读者翻开第一章时心里在等什么"？说不出来 → 没选透。

---

## 原则 2：题材先于一切

**规则：** 题材是 Phase 2 第一步。世界观、角色、写作风格都贴着题材聊。不选题材不往下走。

AI 直接说：
> "我们先把题材定了，再聊世界观和角色。题材定了，后面的讨论才有方向。"

**条件分支：**

| 作者说 | 怎么接 |
|--------|--------|
| "我不知道选什么" | "没关系，说说你脑子里大概是什么方向？古代修仙、现代都市、未来科幻、还是悬疑破案？" |
| "我想写融合的" | "可以。先选主类型，主类型定框架，后续从其他类型追加要素。" |
| "不想选，先聊世界观" | "题材决定世界观往哪个方向聊——仙侠和末世的世界观完全不同。先定题材，世界观才聊得准。" |
| "我看看都有什么" | 打开 index.md 给作者浏览 |

**判断标准：** 作者是否理解题材先定是流程第一步？不清 → 用上面话术解释为什么。

---

## 原则 3：选最接近的，不纠结完美匹配

**规则：** 24 种里挑一个最接近的。没有完美匹配 → 选基类 + 后续叠加。实在不想选 → "暂不选择"也是答案。

展示格式：
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  题材列表（共 24 种）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

都市系（7 种）：都市日常 / 都市脑洞 / 都市种田 / ...

历史系（3 种）：历史古代 / 历史脑洞 / 抗战谍战

玄幻系（2 种）：传统玄幻 / 玄幻脑洞

悬疑系（3 种）：悬疑刑侦 / 悬疑灵异 / 悬疑脑洞

科幻系（1 种）：科幻末世

独立类型（5 种）：西方奇幻 / 东方仙侠 / 男频衍生 / 动漫衍生 / 游戏体育

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
先选主类型，后续可以叠加变体。或者你心里有方向直接说。
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

作者选定后，去 `genre-corpus/{corpus}` 读取该题材的配置，展示给作者确认：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  已选题材：东方仙侠（xianxia）

  文风起点（style_blueprint.voice）：
  叙事者像一个说书人，用词古朴有力……

  满足类型（satisfaction_types）：
  · 境界突破（硬实力跃升）
  · 秘境夺宝（资源争夺战）
  · 打脸反杀（实力碾压后反转评价）

  节奏规则（pacing_rules）：
  · 章节类型：action 40% / setup 25% / development 20% / payoff 15%
  · 张力周期：3-5 章一个小高潮，10 章一个大高潮
  · 铺垫上限：最多连续 2 章铺垫，第 3 章必须有进展

  避免套路（anti_cliches）：
  · 主角跳崖必得奇遇
  · 拍卖会必装逼打脸
  · 入门测试必被看不起

  类型禁忌（genre_taboos）：
  · 不得用现代价值观评判修仙世界的等级制度
  · 禁止将修炼体系写成"只要努力就能成功"的鸡汤

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
确认使用这个配置？还是想调整？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**条件分支（展示后）：**

| 作者说 | 怎么处理 |
|--------|---------|
| "确认" | 写入 settings/genre-setting.md，进入世界观讨论 |
| "XX 不想用" | 去掉对应项后写入 |
| "加 XX 类型的要素" | 读目标类型的 corpus，追加对应 satisfaction_types / pacing_rules |
| "换一个" | 回到列表重新选 |
| "暂不选择" | genre_profile 设为"暂不选择"，跳过，后续随时可回来选 |

**判断标准：** 是否有一个明确的主类型（或"暂不选择"的明确决定）？模棱两可 → 没选完。

---

## 必须包含的内容

写入 `settings/genre-setting.md` 时必须包含以下内容：

| 内容 | 说明 | 来源 |
|------|------|------|
| 选定类型 | 类型 id + label，如 `xianxia — 东方仙侠` | 作者选择 |
| 满足类型 | 3-5 种读者情绪满足类型，每种一句话 | genre_config.satisfaction_types |
| 节奏规则 | 章节类型比例 + 张力周期 + 铺垫上限 | genre_config.pacing_rules |
| 避免套路 | 至少 3 条该类型常见桥段，正文遇到时绕道 | genre_config.anti_cliches |

**选填（暂不选择时没有）：**
- 类型禁忌（genre_taboos）— 写入 genre-setting.md，正文阶段 agent 读取约束
- 文风参考（style_blueprint）— 不写入 genre-setting，留给写作风格讨论用

---

## 验收

| 验收项 | 怎么查 | 不通过处理 |
|-------|--------|-----------|
| 主类型已选定 | genre-profile 有值（类型 id 或"暂不选择"），非空 | 退回继续选 |
| 作者确认过 | 作者至少对展示的配置说过"确认"或"调整后确认" | 退回确认 |
| satisfaction_types ≥ 3 | 数一下有几条 | 不够 3 条 → 退回补充（可追加其他题材的） |
| pacing_rules 完整 | 章节比例 + 张力周期 + 铺垫上限均有值 | 缺项 → 退回从 corpus 补 |
| anti_cliches ≥ 3 | 数一下有几条 | 不够 3 条 → 退回从 corpus 补 |
| 已写入输出文件 | settings/genre-setting.md 存在且内容非空 | 未写入 → 执行写入 |

## 案例

### 案例一：仙侠玄幻

```markdown
# 题材设定

## 选定类型
xianxia — 东方仙侠

## 满足类型
- 境界突破（硬实力跃升）
- 秘境夺宝（资源争夺战）
- 打脸反杀（实力碾压后反转评价）
- 大道争锋（理念冲突 vs 实力对决）

## 节奏规则
- 章节类型：action 40%, setup 25%, development 20%, payoff 15%
- 张力周期：3-5 章一个小高潮，10 章一个大高潮
- 铺垫上限：最多连续 2 章铺垫，第 3 章必须有进展

## 避免套路
- 主角跳崖必得奇遇
- 拍卖会必装逼打脸
- 入门测试必被看不起
- 境界越高越要隐藏实力然后在关键时刻震惊全场
```

### 案例二：都市异能

```markdown
# 题材设定

## 选定类型
urban-brained — 都市脑洞

## 满足类型
- 能力觉醒（获得力量/解锁新能力）
- 隐藏身份（普通人不知道异能世界，主角在两者间周旋）
- 体制 vs 地下（特管处和暗网的灰色地带博弈）
- 日常融合（超能力用在日常场景里的反差趣味）

## 节奏规则
- 章节类型：action 25%, setup 30%, development 25%, payoff 20%
- 张力周期：4-6 章一个小高潮，15 章一个大高潮
- 铺垫上限：最多连续 3 章日常，第 4 章必须有事件推进

## 避免套路
- 主角觉醒就是 S 级
- 特管处全是官僚废物
- 异能者穷得吃不起饭
- 女人负责受伤和被拯救
```
