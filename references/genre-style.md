# 题材设定指南

> 题材是 Phase 2 第一步——定了题材再聊世界观和角色。选完后写入 `settings/genre-setting.md`。
> 共 5 个字段：选定类型 / 满足类型 / 节奏规则 / 避免套路 / 类型禁忌。逐个聊，逐个写。

---

## 先选题材

这步不是写进文件的字段，是聊其他字段的前提——题材定了，才知道该聊哪些方面。

### 怎么聊：让作者说故事，你来判断

不要给作者甩分类列表。作者到这一步肯定有想写的东西，只是不懂术语。直接问：

> "你心里想写的大概是个什么故事？不用管类型分类，说说主角是谁、在什么世界、做什么事就好。"

**不管作者说了什么，你听完后做两件事：**

1. 从作者描述中提取**3 个关键信号**——时代背景、力量体系、核心冲突类型
2. 用下表判断最匹配的类型

| 关键信号 | 匹配类型 |
|---------|---------|
| 现代都市+日常情感+现实生活 | 都市日常 |
| 现代都市+金手指系统+脑洞奇想 | 都市脑洞 |
| 现代都市+种田建设+商战职场 | 都市种田 |
| 现代都市+全民修炼+灵气复苏 | 都市高武 |
| 现代都市+修仙/修真体系 | 都市修真 |
| 现代都市+战神归来+赘婿翻身 | 战神赘婿 |
| 古代背景+种田/朝堂/争霸 | 历史古代 |
| 古代背景+金手指+脑洞 | 历史脑洞 |
| 抗战时期+军事/谍战 | 抗战谍战 |
| 异世大陆+高武+废柴逆袭 | 传统玄幻 |
| 异世大陆+脑洞设定 | 玄幻脑洞 |
| 现代+探案/推理/刑侦 | 悬疑刑侦 |
| 现代+灵异/风水/恐怖 | 悬疑灵异 |
| 诡异复苏+规则怪谈+惊悚游戏 | 悬疑脑洞 |
| 末世/丧尸/星际/机甲/未来科技 | 科幻末世 |
| 西方世界+魔法/斗气/奇幻 | 西方奇幻 |
| 东方世界+修仙/修道/修真 | 东方仙侠 |
| 影视剧同人改编 | 男频衍生 |
| 游戏/动漫/二次元同人 | 动漫衍生 |
| 网游/竞技/体育/世界游戏化 | 游戏体育 |

**判断后展示给作者确认：**

> "听起来你写的是 **{类型}**——{一句理由}。对吗？"

✅ 正确：
> 作者："我想写一个普通上班族某天突然能看到别人头顶的死亡倒计时，他试着去救这些人，结果发现倒计时越改越短。"
> → 信号提取：现代都市（上班族）+ 金手指系统（死亡倒计时）+ 救人反噬
> → 判断：**都市脑洞（urban-brained）**
> → 确认："听起来你写的是都市脑洞——现代都市背景，主角获得特殊能力，但这个能力有代价。对吗？"

❌ 错误：
> 作者："我想写个故事。" → 继续问："主角是什么样的人？故事发生在什么地方？古代还是现代？"

**条件分支：**

| 作者说 | 怎么接 |
|--------|--------|
| "对，就是这个" | 进入字段 1，写入选定类型 |
| "不太对，更像 XX" | 按作者修正重新判断，或者直接手动选 |
| "还是不知道算什么类型" | "没关系，说说你故事里有没有超自然力量？古代还是现代？——这些定了就好分类了。" |
| "不想选类型" | "题材定了世界观才好聊——仙侠和末世的世界观完全不同。先定题材，世界观才聊得准。""暂不选择"也是有效答案，后续随时补选。 |

**判断标准：** 作者明确说了一个类型（或"暂不选择"）？模棱两可 → 继续聊。

**不需要读外部文件。** 下面每个字段都内联了多题材示例，直接参考。

---

## 字段 1：选定类型

**写什么：** 类型 id + 中文名，一行。

```
xianxia — 东方仙侠
```

id 用小写英文连字符，中文名用作者认可的表述。如果选融合类型，标注主类型 + 叠加要素：

```
urban-brained — 都市脑洞（叠加悬疑刑侦要素）
```

**判断标准：** 读完能知道这是什么题材、融合了什么要素。

---

## 字段 2：满足类型

**写什么：** 3-5 种读者想从这个题材获得的"满足感"，每种一句话说清核心吸引力。

题材不同，读者期待不同。不要列抽象标签（"爽"），要说明具体怎么爽。不满足读者期待的类型 → 保留但要降低占比。

**怎么聊：**
> "你觉得读者看你这个题材，最想得到什么？——打脸升级？烧脑推理？人情世故？还是别的？"

如果作者说不上来，给题材一个起点参考：

| 题材 | 常见的满足类型 |
|------|--------------|
| 东方仙侠 | 境界突破、秘境夺宝、打脸反杀、大道争锋 |
| 都市脑洞 | 能力觉醒、隐藏身份、体制 vs 地下、日常融合 |
| 悬疑刑侦 | 线索推理、真相反转、猫鼠游戏、罪案心理 |
| 历史古代 | 朝堂博弈、军事征战、改革建设、人情世故 |

✅ 正确：
> 境界突破（硬实力跃升）——每次突破给读者"又变强了"的确定回报。

❌ 错误：
> 爽/热血/刺激——太抽象，不知道具体怎么"爽"。

**各题材示例（直接参考，不用读外部文件）：**

| 题材 | 满足类型示例 |
|------|-------------|
| 东方仙侠 | 境界突破、秘境夺宝、打脸反杀、大道争锋 |
| 都市脑洞 | 能力觉醒、隐藏身份、体制 vs 地下、日常融合 |
| 悬疑刑侦 | 线索推理、真相反转、猫鼠游戏、罪案心理 |
| 历史古代 | 朝堂博弈、军事征战、改革建设、人情世故 |

**判断标准：** 每一条满足类型能否对应一个具体的剧情桥段？不能 → 太抽象。

## 字段 3：节奏规则

**写什么：** 三条规则——章节类型比例、张力周期、铺垫上限。

### 章节类型比例

一章的内容分几种类型，各占多少比例。不是固定公式，是让 agent 写章节时有节奏意识。

| 类型 | 含义 | 占比范围 |
|------|------|---------|
| 行动 | 冲突、战斗、对抗——读者被吊着看 | 25%-40% |
| 铺垫 | 信息铺设、氛围营造、日常过渡 | 20%-30% |
| 发展 | 角色成长、关系推进、资源积累 | 20%-25% |
| 收获 | 揭晓答案、反转、高潮兑现 | 15%-20% |

不同类型题材比例不同——悬疑推理的铺垫比例高，仙侠爽文的行动比例高。

**怎么聊：**
> "你的故事节奏偏快还是偏慢？——几章得来一次小高潮？能接受连续几章铺垫不推进？"

### 张力周期

多少章一个小高潮、多少章一个大高潮。

**规律：**
- 爽文：3-5 章小高潮，10 章大高潮
- 悬疑：4-6 章小高潮，15 章大高潮
- 日常：5-8 章小高潮，20 章大高潮

### 铺垫上限

"最多连续几章没有进展"。超过这个上限读者会弃书。

**范围参考：** 快节奏 2 章，中等 3 章，慢热 4 章。

✅ 正确（仙侠快节奏）：
> - 章节类型：行动 40%, 铺垫 25%, 发展 20%, 收获 15%
> - 张力周期：3-5 章一小高潮，10 章一大高潮
> - 铺垫上限：最多连续 2 章铺垫，第 3 章必须有进展

✅ 正确（悬疑慢热）：
> - 章节类型：行动 25%, 铺垫 30%, 发展 25%, 收获 20%
> - 张力周期：4-6 章一小高潮，15 章一大高潮
> - 铺垫上限：最多连续 3 章铺垫，第 4 章必须有事件推进

❌ 错误：
> 节奏适中，该快则快该慢则慢——说了等于没说。

**判断标准：** agent 读了会不会"知道了但不知道怎么执行"？会 → 不够具体。

## 字段 4：避免套路

**写什么：** 这个题材读者看腻了的桥段。≥3 条。

套路不是不能写，而是不能无意识地写——写之前要知道"这是个套路"，然后决定是颠覆它、绕过它、还是承认它但写得更好。

**怎么聊：**
> "这个题材有什么桥段你不想写的？——就是那种一看开头就知道结尾的老套路。"

作者可能列出来的东西需要加工：

| 作者说了 | 写成 |
|---------|------|
| "不想写老套的拍卖会打脸" | 拍卖会必装逼打脸 |
| "不要跳崖奇遇" | 主角跳崖必得奇遇 |
| "不想主角上来就无敌" | 主角觉醒就是 S 级 |

✅ 正确：
> 主角跳崖必得奇遇——真跳崖了未必有奇遇，反而可能是埋伏。

❌ 错误：
> 不要写套路——太宽泛，什么算套路？

**各题材示例（直接参考）：**
- 仙侠：跳崖得奇遇、拍卖会装逼打脸、入门测试被看不起、境界高隐藏实力震惊全场
- 都市脑洞：主角觉醒就是 S 级、特管处全是废物、异能者穷得吃不起饭、女性角色只负责受伤被救
- 悬疑刑侦：第一目击者就是凶手、最不可能的凶手、天才侦探一眼看穿、最终boss总是熟人

**判断标准：** 每一条能不能写进一个 5 分钟视频的"网络小说套路吐槽"里？能 → 合格。

---

## 字段 5：类型禁忌

**写什么：** 这个题材绝对不能出现的东西——写了读者弃书、类型不成立。

禁忌不是可选的"尽量避开"，是写进去整本书变味的东西。

**常见禁忌分三类：**

| 类别 | 含义 | 例子 |
|------|------|------|
| 体系崩坏 | 自洽性被打破 | 战力忽高忽低、修炼一步登天、法宝随心所欲 |
| 题材错位 | 出现题材不该有的东西 | 仙侠世界讲人人平等、古代出现现代价值观 |
| 承诺失信 | 核心体验被破坏 | 推理小说靠超自然破案、末世文没有生存压力 |

**怎么聊：**
> "你觉得这个题材什么东西绝对不能写？——写了读者会骂'这写的什么玩意'那种。"

如果作者说不上来，顺着已选的题材举例询问：

> "仙侠文里战力崩坏是常见雷区——你介意吗？还有现代价值观乱入修真界呢？"

✅ 正确（仙侠）：
> - 禁止力量体系崩坏——战力不能为剧情方便忽高忽低
> - 禁止现代价值观乱入修真界

❌ 错误：
> 不要写得太离谱——什么算离谱？谁定义的？

**各题材示例（直接参考）：**

| 题材 | 常见禁忌 |
|------|---------|
| 东方仙侠 | 战力崩坏、修炼一步登天、现代价值观乱入、法宝随意设定、天劫走过场 |
| 都市脑洞 | 异能无代价、政府完全不知情、主角全知全能、体系逻辑不自洽 |
| 悬疑刑侦 | 超自然破案（非灵异题材）、凶手动机牵强、线索凭空出现、反派降智 |
| 历史古代 | 现代思想穿越、科技跳跃过快、历史人物纸片化、金手指过强无制约 |

**判断标准：** 作者能不能想象"违反这条禁忌写出来是什么样"？不能 → 太抽象。

## 写入文件

写入 `settings/genre-setting.md`，Markdown 格式。完整格式参考：

```markdown
# 题材设定

## 选定类型
xianxia — 东方仙侠

## 满足类型
- 境界突破（硬实力跃升）
- 秘境夺宝（资源争夺战）
- 打脸反杀（实力碾压后反转评价）
- 大道争锋（理念冲突 vs 实力对决）

## 节奏规则
- 章节类型：行动 40%, 铺垫 25%, 发展 20%, 收获 15%
- 张力周期：3-5 章一个小高潮，10 章一个大高潮
- 铺垫上限：最多连续 2 章铺垫，第 3 章必须有进展

## 避免套路
- 主角跳崖必得奇遇
- 拍卖会必装逼打脸
- 入门测试必被看不起

## 类型禁忌
- 禁止力量体系崩坏
- 禁止修炼一步登天
- 禁止现代价值观乱入修真界
```

## 验收

### 五个字段逐个检查

| 字段 | 怎么算够 | 怎么算不够 |
|------|---------|-----------|
| 选定类型 | 有具体的类型 id（或"暂无选择"），非空 | 模棱两可，或写"类型待定"但没说明方向 |
| 满足类型 | ≥ 3 条，每条约一句话说清具体吸引力 | 抽象标签（"爽""热血"），或条数不够 |
| 节奏规则 | 章节比例 + 张力周期 + 铺垫上限均有具体值 | 缺项，或写"该快则快"等不可执行的描述 |
| 避免套路 | ≥ 3 条，每条约是一个具体的桥段 | 写得太宽泛（"不要套路"），或不够 3 条 |
| 类型禁忌 | 有（至少 2 条），且能想象违反它是啥样 | 写得太抽象，或空着 |

### 总体检查

- [ ] 五个字段全部有值（选定类型可为"暂无选择"，但需有方向说明）？
- [ ] 每条满足类型能对应具体剧情桥段？
- [ ] 节奏规则的三条子项都有具体值？
- [ ] 避免套路的每条是"一眼能认出的老桥段"？
- [ ] 类型禁忌每条有"违反后会怎样"的后果说明？

### 参考案例

指南中每个字段已内联了多题材示例（仙侠/都市/悬疑/历史），不需要额外读外部文件。如果你需要某题材的完整配置参考（含 story_arc_templates、prompt_segment 等未在指南中覆盖的字段），可以打开 `references/genre-example/{类型id}.md` 查看——但这不在正常对话流程内。
