# 题材选择指南

> 题材是 Phase 2 第一步——定了题材再聊世界观和角色。选完后写入 `settings/genre-setting.md`。
> 流程：展示列表 → 作者选 → 读 corpus 配置 → 展示确认 → 写入文件。

---

## Step 1：展示题材列表

先读 `genre-corpus/index.md`，给作者展示 24 种题材：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  题材列表（共 24 种）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

都市系（7 种）：都市日常 / 都市脑洞 / 都市种田 / 都市高武 / 都市修真 / 战神赘婿
历史系（3 种）：历史古代 / 历史脑洞 / 抗战谍战
玄幻系（2 种）：传统玄幻 / 玄幻脑洞
悬疑系（3 种）：悬疑刑侦 / 悬疑灵异 / 悬疑脑洞
科幻系（1 种）：科幻末世
独立类型（5 种）：西方奇幻 / 东方仙侠 / 男频衍生 / 动漫衍生 / 游戏体育

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
先选主类型，后续可以叠加变体。或者你心里有方向直接说。
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**条件分支：**

| 作者说 | 怎么接 |
|--------|--------|
| "我不知道选什么" | "没事，说说你脑子里大概是什么方向？古代修仙、现代都市、未来科幻、还是悬疑破案？" |
| "我想写融合的" | "可以。先选主类型，主类型定框架，后续从其他类型追加要素。" |
| "不想选" | "题材定了世界观才好聊——仙侠和末世的世界观完全不同。先定题材，世界观才聊得准。""暂不选择"也是有效答案，后续随时补选。 |
| "我看看都有什么" | 已经展示了列表，让他挑 |

**判断标准：** 作者是否明确说了一个类型（或"暂不选择"）？模棱两可 → 继续聊。

---

## Step 2：展示题材配置

作者选定后，去 `genre-corpus/{corpus}` 读取该题材的全部配置，展示给作者确认：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  已选题材：东方仙侠（xianxia）

  文风起点（style_blueprint.voice）：
  叙事者像一个说书人，用词古朴有力……

  满足类型（satisfaction_types）：
  · 境界突破（硬实力跃升）
  · 秘境夺宝（资源争夺战）
  · 打脸反杀（实力碾压后反转评价）

  节奏规则（pacing_rules）：
  · 章节类型：action 40% / setup 25% / development 20% / payoff 15%
  · 张力周期：3-5 章一个小高潮，10 章一个大高潮
  · 铺垫上限：最多连续 2 章铺垫，第 3 章必须有进展

  避免套路（anti_cliches）：
  · 主角跳崖必得奇遇
  · 拍卖会必装逼打脸
  · 入门测试必被看不起

  类型禁忌（genre_taboos）：
  · 禁止力量体系崩坏
  · 禁止修炼一步登天
  · 禁止现代价值观乱入修真界

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
确认使用这个配置？还是想调整？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**条件分支：**

| 作者说 | 怎么处理 |
|--------|---------|
| "确认" | 进 Step 3，写入文件 |
| "XX 不想用" | 去掉对应项后进 Step 3 |
| "加 XX 类型的要素" | 读目标类型的 corpus，追加对应 satisfaction_types / pacing_rules |
| "换一个" | 回 Step 1 重新选 |
| "暂不选择" | genre_profile 设为"暂不选择"，跳过。后续随时补选 |

---

## Step 3：写入输出文件

写入 `settings/genre-setting.md`。格式参考下方案例，必须包含以下内容：

| 内容 | 说明 | 来源 |
|------|------|------|
| 选定类型 | 类型 id + label，如 `xianxia — 东方仙侠` | 作者选择 |
| 满足类型 | 3-5 种，每种一句话 | corpus 的 genre_config.satisfaction_types |
| 节奏规则 | 章节类型比例 + 张力周期 + 铺垫上限 | corpus 的 genre_config.pacing_rules |
| 避免套路 | ≥3 条 | corpus 的 genre_config.anti_cliches |
| 类型禁忌 | 全部写入 | corpus 的 genre_taboos |

> **style_blueprint 不写入 genre-setting.md**——留给稍后的写作风格讨论用。

### 案例一：仙侠玄幻（写入后的成品格式）

```markdown
# 题材设定

## 选定类型
xianxia — 东方仙侠

## 满足类型
- 境界突破（硬实力跃升）
- 秘境夺宝（资源争夺战）
- 打脸反杀（实力碾压后反转评价）
- 大道争锋（理念冲突 vs 实力对决）

## 节奏规则
- 章节类型：action 40%, setup 25%, development 20%, payoff 15%
- 张力周期：3-5 章一个小高潮，10 章一个大高潮
- 铺垫上限：最多连续 2 章铺垫，第 3 章必须有进展

## 避免套路
- 主角跳崖必得奇遇
- 拍卖会必装逼打脸
- 入门测试必被看不起
- 境界越高越要隐藏实力然后在关键时刻震惊全场

## 类型禁忌
- 禁止力量体系崩坏
- 禁止修炼一步登天
- 禁止现代价值观乱入修真界
- 禁止法宝能力随意设定
- 禁止天劫走过场
```

### 案例二：都市异能（写入后的成品格式）

```markdown
# 题材设定

## 选定类型
urban-brained — 都市脑洞

## 满足类型
- 能力觉醒（获得力量/解锁新能力）
- 隐藏身份（普通人不知道异能世界，主角在两者间周旋）
- 体制 vs 地下（特管处和暗网的灰色地带博弈）
- 日常融合（超能力用在日常场景里的反差趣味）

## 节奏规则
- 章节类型：action 25%, setup 30%, development 25%, payoff 20%
- 张力周期：4-6 章一个小高潮，15 章一个大高潮
- 铺垫上限：最多连续 3 章日常，第 4 章必须有事件推进

## 避免套路
- 主角觉醒就是 S 级
- 特管处全是官僚废物
- 异能者穷得吃不起饭
- 女人负责受伤和被拯救
```

## 验收（写完后逐项检查）

| 验收项 | 怎么查 | 不通过处理 |
|-------|--------|-----------|
| 主类型已选定 | genre-profile 有值（类型 id 或"暂不选择"），非空 | 退回继续选 |
| 作者确认过 | 作者至少对展示的配置说过"确认"或"调整后确认" | 退回确认 |
| satisfaction_types ≥ 3 | 数一下有几条 | 不够 → 退回补充（可从其他 corpus 追加） |
| pacing_rules 完整 | 章节比例 + 张力周期 + 铺垫上限均有值 | 缺项 → 退回从 corpus 补 |
| anti_cliches ≥ 3 | 数一下有几条 | 不够 → 退回从 corpus 补 |
| genre_taboos 已写入 | 有"类型禁忌"章节，非空 | 未写入 → 从 corpus 读取写入 |
| 已写入输出文件 | settings/genre-setting.md 存在且内容非空 | 未写入 → 执行写入 |
