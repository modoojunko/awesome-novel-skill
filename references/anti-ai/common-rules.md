# 反AI通用规则

> 所有题材共用。正文写作必读。

## 一、疲劳词阈值

| 类别 | 词 | 单章阈值 | 替代策略 |
|------|---|---------|---------|
| 副词 | 突然/忽然/猛然 | ≤3次 | 删掉，事件本身已传达"突然" |
| 副词 | 竟然/居然 | ≤2次 | 让读者自己判断 |
| 副词 | 渐渐/逐渐 | ≤2次 | 用变化细节替代 |
| 副词 | 依旧/仍然 | ≤2次 | 用状态描写替代 |
| 动词 | 露出（表情） | ≤3次 | 用具体笑法替代 |
| 动词 | 陷入（沉默） | ≤2次 | 写沉默后的具体行为 |
| 动词 | 涌起/泛起（情绪） | ≤2次 | 用身体反应替代 |
| 连接词 | 然而/不过 | ≤4次 | 删掉，句子直接碰撞 |
| 连接词 | 于是 | ≤3次 | 删掉，因果自现 |
| 连接词 | 与此同时 | ≤1次 | 直接换场景开头 |
| 身体模板 | 瞳孔一缩/身体一僵 | ≤2次 | 用具体场景替代 |
| 身体模板 | 倒吸一口凉气/呼吸一滞 | ≤1次 | 用呼吸变化替代 |

## 二、句式规则

| 规则 | 说明 |
|------|------|
| 禁止连续4句相同主语 | 变换主语和句式 |
| 禁止"不是X而是Y"句式 | ≥4次/章 → 删掉否定分句 |
| 禁止"让/令/使"句式 | 删掉让，主语直接行动 |
| "了"字净化 | 连续3句5个以上"了" → 改写2-3句 |
| 禁止"当...时"句式 | 拆成独立动作句 |

## 三、元叙事禁止（发现即删）

| 类型 | 词 |
|------|---|
| 章节定位词 | "本章""下一章""故事还在继续" |
| 学术腔 | "综上所述""总而言之""显而易见" |
| 模糊词 | "似乎/也许/大概/仿佛" ≤2次/500字 |

## 四、情绪外化规则

| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| "他意识到/她感到"直述情绪 | 用动作、环境、对话外化 |
| AI套话、强行升华 | 停在一个画面/动作上 |
| 分析性结论句 | 删掉，让画面说话 |
