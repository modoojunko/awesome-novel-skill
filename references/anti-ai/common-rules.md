# 反AI通用规则

> 所有题材共用。正文写作必读。

## 一、疲劳词阈值

| 类别 | 词 | 单章阈值 | 替代策略 |
|------|---|---------|---------|
| 副词 | 突然/忽然/猛然 | ≤3次 | 删掉，事件本身已传达"突然" |
| 副词 | 竟然/居然 | ≤2次 | 让读者自己判断 |
| 副词 | 渐渐/逐渐 | ≤2次 | 用变化细节替代 |
| 副词 | 依旧/仍然 | ≤2次 | 用状态描写替代 |
| 动词 | 露出（表情） | ≤3次 | 用具体笑法替代 |
| 动词 | 陷入（沉默） | ≤2次 | 写沉默后的具体行为 |
| 动词 | 涌起/泛起（情绪） | ≤2次 | 用身体反应替代 |
| 连接词 | 然而/不过 | ≤4次 | 删掉，句子直接碰撞 |
| 连接词 | 于是 | ≤3次 | 删掉，因果自现 |
| 连接词 | 与此同时 | ≤1次 | 直接换场景开头 |
| 身体模板 | 瞳孔一缩/身体一僵 | ≤2次 | 用具体场景替代 |
| 身体模板 | 倒吸一口凉气/呼吸一滞 | ≤1次 | 用呼吸变化替代 |

## 二、句式规则

| 规则 | 说明 |
|------|------|
| 禁止连续4句相同主语 | 变换主语和句式 |
| 禁止"不是X而是Y"句式 | ≥4次/章 → 删掉否定分句 |
| 禁止"让/令/使"句式 | 删掉让，主语直接行动 |
| "了"字净化 | 连续3句5个以上"了" → 改写2-3句 |
| 禁止"当...时"句式 | 拆成独立动作句 |

## 三、元叙事禁止（发现即删）

| 类型 | 词 |
|------|---|
| 章节定位词 | "本章""下一章""故事还在继续" |
| 学术腔 | "综上所述""总而言之""显而易见" |
| 模糊词 | "似乎/也许/大概/仿佛" ≤2次/500字 |

## 四、情绪外化规则

| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| "他意识到/她感到"直述情绪 | 用动作、环境、对话外化 |
| AI套话、强行升华 | 停在一个画面/动作上 |
| 分析性结论句 | 删掉，让画面说话 |

## 五、重复检测

| 规则 | 说明 |
|------|------|
| 禁止连续3段相同开头 | 每段首句主语/场景不能重复 |
| 禁止连续5句相同句式 | 陈述/短句/对话交替 |
| 禁止连续2段描写同一物 | 景物/人物/物品不重复描写 |
| 禁止同一情绪词重复 | 紧张/开心/难过等情绪词单章≤3次 |
| 禁止相同比喻复用 | 用过的比喻/意象本章不再用 |

## 六、视角规则

| ❌ 禁止 | ✅ 要求 |
|--------|--------|
| 上帝视角（读者代入感差） | 聚焦主角视角 |
| 视角在配角间跳来跳去 | 主角一定在场 |
| 配角戏份超过主角 | 配角剧情一笔带过 |

**前期（10万字内）**：
- 开篇围绕主角一人
- 其他人发生的事情一笔带过或听别人提

**后期（10万字后）**：
- 读者代入感建立后可少量添加
- 控制在几百字篇幅
- 不宜大篇幅讲配角

## 七、剧情转折规则

| 问题 | 根因 | 修复方法 |
|------|------|---------|
| 转折生硬 | 缺"心理过渡" | 加内心独白，为主角行为做预期解释 |
| 冲突不够 | 双方没有"不能退让的理由" | 给冲突双方各加"我必须这么做"的强烈动机 |

**冲突规则：**
- 失败的代价越具象（死/失去什么），读者越能理解
- 双方都要有"必须得到"的强烈信念

## 八、节奏与铺垫

| 规则 | 说明 |
|------|------|
| 铺垫不够 = 故事没劲 | 前期打压引申出期待感 |
| 无效剧情删 | 完全跟主角没关系的剧情 = 无效 |
| 详略得当 | 配角支线跟着剧情走，服务主角 |

**节奏模式（番茄标准）：压 → 缓解 → 再压 → 爆发**
1. 先压（生存环境/打压）
2. 缓解（希望/日常/温情）
3. 再压（更深的绝望）
4. 爆发（正义伸张/爽点释放）

## 九、对话节奏

| 规则 | 说明 |
|------|------|
| 连续对话不超过3句 | 对话之间必须有动作/沉默/环境/内心 |
| 减少"了"字 | "了"字多了强行让读者认同作者 |
| 减少转折词 | "虽然…但是…" "然而" "却"用多了 = AI味 |
