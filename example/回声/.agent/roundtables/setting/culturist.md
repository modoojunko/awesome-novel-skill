# 文化师问答

## 批量提问（第1轮）

Q1: 天后宫在临港老城的信仰地位如何？——混合了妈祖、土地公、山神？
答: 妈祖主神，偏殿土地公和"山神"（清末舍身救渔民的船工被民间封神）。年轻一代基本不拜，春节和妈祖诞辰商家捐香火。日常由老年妇女（"妈祖婆"）维持。

Q2: 1994年白岛失踪案在临港民间怎么传的？有没有都市传说？
答: 流传最广：病人被"白岛上的东西"带走。"白岛每隔十二年收一个人"——1994年、2006年、2018年。渔民非必要不上岛，尤其农历七月。另有版本说病人知道了疗养院秘密被"处理"了。

Q3: 北郊山洞群当地叫什么？有没有民间用途或传闻？
答: "鬼喊洞"，因风吹洞穴发出啸叫声得名。北郊20公里，地下暗河系统。七月半放河灯。主洞有1958年勘探标记和60-70年代涂鸦。有一个支洞尽头人工封堵痕迹，陈远征注意到但未深究。

Q4: 青石街居民以什么群体为主？邻里关系如何？
答: 中老年原住民为主，年轻人搬走。民宿奶茶店替代老店铺。表面客气，涉及敏感话题闭嘴。"茶桌情报网"——每天早上天后宫茶摊的老头们掌握全城信息。

Q5: 废弃码头改文创园后，老码头工人去哪了？有冲突吗？
答: 转物流或退休。微妙的怨恨——"我们的过去当布景"。回声酒吧是核心据点，老板是前记者。

Q6: 临港的阶层地图？
答: 新城白领+开发区投资者顶层，老城原住民+码头工人底层，文创园艺术家中间但被两边看不上。灰色地带：港区临时工（码头散工、快艇接驳人），警察不太管。

Q7: 临港的城市性格？
答: "表面开放、内里封闭"。礼貌不交底。"不该问的别问"是社会默契。突破入口：茶摊、酒桌、渔友关系。

## 追问（第2轮）

Q8: 白岛1994、2006、2018年三次事件的异同？民间有没有规律清单？
答: 1994年江成母亲（护士）。2006年男游客迷路2天后找回（"收人未遂"）。2018年偷猎者火光。无明确共同点。民间串联成"白岛三色光"——白衣、蓝衣、鬼火。

Q9: 陈远征注意到鬼喊洞支洞封堵的细节？当地老人知道封堵前用途吗？
答: 水泥砂浆掺海沙，像仓促施工，封堵体不厚。解放前是"躲兵洞"，1958年勘探队扩挖，后不知何时被封，无人承认。

Q10: 妈祖婆群体对白岛传说态度？
答: 阿珍婆（80多岁）1994年在码头见过苏榕等船——穿便服拎小包，不像要出远门。从未对人说过此事。

## 摘要
- 信仰体系: 妈祖+土地公+民间封神（山神）
- 都市传说: 白岛十二年收一人、白岛三色光
- 关键地点传闻: 鬼喊洞（躲兵洞→封堵）、回声酒吧（前记者老板）
- 情报节点: 天后宫茶摊、回声酒吧、港口调度室
- 关键伏笔: 阿珍婆见过苏榕等船
