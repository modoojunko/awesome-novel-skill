# 角色师问答

## 批量提问（第1轮）

Q1: 主角是谁？
答: 江成，36岁，临港市局刑侦支队副支队长。临港出生长大，警校毕业回来干了14年刑警。母亲苏榕1994年是白岛疗养院护士，在岗期间失踪。父亲说"跟人跑了"，江成不信，一直在暗中查。只有周铭知道。

Q2: 核心搭档？
答: 林筱，28岁，技术科法医（老刘徒弟），省厅培训回来。刑侦+法医跨科室搭档。冷静、观察力强，和江成直觉型互补。合作关系加朋友。林筱的父亲当年是疗养院医生。

Q3: 主角与白岛案的私人关系？
答: 江成的母亲苏榕就是1994年失踪的人。他12岁时偷去过白岛被送回来。考警校、回临港、进刑侦——这个案子是他从警的深层原因。只有周铭知道这个背景。

Q4: 反派设计？
答: 贯穿线——白岛1994年案与当前新案件（曹斌案）有关联。幕后人是一个利益联盟（当年参与掩盖的相关方），曹斌案是这个联盟出现裂痕的结果。

Q5: 政治立场？
答: 不选边。赵海涛有知遇之恩但对白岛案态度让江成无法信任。周铭知道江成背景是潜在盟友但根基浅。江成靠案件进展说话，深入调查后将被迫选择。

Q6: 主角弱点？
答: 过度投入——查到和白岛相关就停不下来，会突破正常程序。对母亲失踪的执念造成判断盲区——容易相信"母亲是被害"的线索。陈远征评价："他太想破这个案子了。"

## 追问（第2轮）

Q7: 母亲身份澄清——护士还是病人？
答: 母亲苏榕是白岛疗养院的**护士**，不是病人。她上白岛前和江成父亲关系已不好。某天夜班后失踪（个人物品都在，班次记录正常），院方结论"自行离岗→走失"。失踪前对同事说过"有些事知道了不好"。

Q8: 周铭、陈远征、赵海涛三人的具体职务？
答: 赵海涛：临港市副市长兼公安局局长，三级警监。周铭：临港市公安局副局长（分管刑侦），二级警督。陈远征：省厅刑侦总队二处副处长（积案督导），一级警督。

Q9: 当前文创园案件的具体信息？
答: 死者曹斌，42岁，回声酒吧合伙人兼歌手。死在文创园B区废弃仓库钢架下，疑点：无攀爬痕迹、死亡当晚和一未注册号码通话47秒匿名公用电话报警。指向白岛：林筱发现曹斌左手腕旧疤痕，江成认出和母亲遗物笔记中描述的"白岛疗养院约束带疤痕"相似。曹斌经营酒吧前曾在私人调查公司工作过。

## 摘要
- 主角: 江成，36岁，刑侦支队副支队长
- 核心驱动: 母亲苏榕（白岛疗养院护士，1994年失踪）
- 搭档: 林筱，28岁法医（父亲曾是疗养院医生）
- 权力三角: 赵海涛（本地派局长）/ 周铭（空降派副局长）/ 陈远征（省厅积案督导）
- 反派: 利益联盟，曹斌案为裂痕
- 主角弱点: 过度投入，判断盲区
- 关键伏笔: 曹斌约束带疤痕、阿珍婆目击
