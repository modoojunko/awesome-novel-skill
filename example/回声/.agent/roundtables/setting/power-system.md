# 力量体系师问答

## 批量提问（第1轮）

Q1: 临港市局刑科所鉴定能力如何？
答: 市局能做常规DNA（STR），复杂样本送省厅3-5天。毒物常规可做，新型罕见毒物送省厅。微量物证基础对比可做，高级分析送省厅。新城区/主干道高清数字（2018后），老城巷区/北郊标清模拟，山洞/码头老区大量盲区。

Q2: 凶手比警方多掌握什么信息？
答: 核心优势是对临港地下空间的熟悉度——知道一条从老城区某处通往北郊暗河系统的"老路"（当年勘探队留下的），从未出现在官方地图上。此外可能掌握1994年案的内部信息。

Q3: 白岛案物证保存状况？
答: 原始卷宗不完整，现场照片底片"丢失"。物证袋只有衣物和拖鞋，DNA降解严重。马国良"记不清了"。疗养院档案在铁路分局档案室，调取需正式手续。

Q4: 监控死角？
答: 青石街巷区基本无监控（街口一个标清探头，盲区大）。废弃码头无公安监控，仅文创园自装网络摄像头。北郊山洞群无监控。白岛无监控。

Q5: 通讯与数据痕迹？
答: 可调通话记录（6个月内）和基站定位，需分局以上领导批。社交媒体数据需省厅网安协调，较慢。老城区部分区域4G弱。

Q6: 尸检话语权？
答: 市局一名在编法医（老刘，25年），一名助理。老刘和赵海涛关系好，不受行政干预。复检程序：家属/上级要求或新证据，分局以上批。

Q7: 民间信息网？
答: 核心节点：天后宫茶摊、回声酒吧（前记者老板）、港口货运站调度室。社区民警有线人但只提供表层信息。主角需赢得非正式关系才能拿到"地下情报"。

## 摘要
- 鉴证层级: 市局常规→省厅复杂，新旧城区技术差距
- 凶手优势: 地下暗河老路+1994案内情
- 物证障碍: 底片丢失/DNA降解/证人回避
- 关键节点: 回声酒吧、茶摊、调度室
