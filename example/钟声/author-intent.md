# 作者意图

## 核心主题

一个背负过去的人如何在追寻真相的过程中面对自己的愧疚。真相不一定带来解脱，但必须被看见。

## 终局方向

陆征最终发现三年前悬案的真相与警局内部有关——不是某个坏人的阴谋，而是一连串每个人都觉得自己"没办法"的选择叠加在一起的结果。他必须决定：揭发还是沉默。

## 写作信条

- 每件事都有代价，好人做坏事和坏人做好事都应该有合理动机
- 不让角色轻易获得"道德正确"的位置
- 城市是一个角色——环境不是背景板，是人物状态的一部分
- 悬疑的核心是人，不是诡计

## 绝不妥协

- 不出现"最后一刻证据从天而降"
- 不出现纯粹邪恶或纯粹正义的角色
- 不说教——不让任何角色成为作者的传声筒
- 不让陆征变成一个无所不能的神探

## 长期伏笔池

- 三年前陆征离开警队的真正原因（涉及一起办案失误）
- 江城地下非法借贷网络
- 方岩与陆征之间未解决的旧事
- 钟声桥的意象——每次重大转折前听到钟声
