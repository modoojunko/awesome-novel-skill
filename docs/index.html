不只是"代写"，是真正的协作

每一步都和你讨论确认，你掌舵，AI 划桨——比起工具，它更像一个有编辑经验的小说搭档

🌍

世界观搭建

地理、政治、文化、特殊规则……像和编辑聊天一样，逐项讨论并整理成结构化的世界设定文档。

👤

角色深度塑造

性格、价值观、能力体系、成长经历——每个角色独立建档，角色状态随剧情推进自动更新。

📋

章节精细规划

先分卷再分章，逐章讨论核心情节、出场角色、场景安排，章纲确认后才动笔。

🛡️

去 AI 味检测

自动扫描机器腔——句式开头重复、身体反应模板化、公式化副词、"不是X而是Y"分析句式，标出来让你改。

🪝

伏笔全生命周期追踪

从埋伏笔到回收，每个钩子的状态自动追踪——pending → mentioned → resolved → abandoned，该填坑时提醒你。

📊

节奏健康检查

连续高压超过 3 章、连续平淡超过 2 章，自动提醒调整叙事节奏。

6 阶段协作工作流

每一步都有明确的分工——Agent 引导讨论、整理信息，你做所有关键决定

1

初始化

创建项目骨架，搭好全部目录结构和配置模板

2

设定

讨论世界观、角色、写作风格，选择题材画像

3

故事线拆分

逐卷逐章讨论章纲，同步管理伏笔和钩子

4

提示词生成

视角转换 + 三层技法分发，生成高质量 prose 提示词

5

正文写作

AI 执笔生成正文，通过 15 项质量检测 + 10 维深度评审后才交付

6

归档

正文存档，自动更新角色状态、伏笔状态、项目索引

一分钟安装

选择你使用的 AI 编程工具，复制命令到终端执行

$ git clone https://github.com/modoojunko/awesome-novel-skill.git && cd awesome-novel-skill && ./install.sh claude-code
$ git clone https://github.com/modoojunko/awesome-novel-skill.git && cd awesome-novel-skill && ./install.sh hermes
$ git clone https://github.com/modoojunko/awesome-novel-skill.git && cd awesome-novel-skill && ./install.sh openclaw

安装完成后，打开终端，在你想写小说的目录下启动 AI 工具，说一句 "我想写一本玄幻小说，帮我开始" 即可。

现在就动笔

不管你是第一次写小说还是老作者，Novel Agent 帮你把想法变成章节——安装只需一分钟。
