# Best Practices — 设定阶段

讨论世界观、角色、写作风格时参考以下案例和提问角度。

## 选题与灵感

引导作者从具体事件中提取素材，而非空谈题材标签。

> **提问：** "最近有没有看到让你印象深刻的新闻？或者身边发生的事？"
> **案例：** "深圳卫健委"公众号的爆款短剧——从医患纠纷中提取冲突。

- 小人物牵出大事件，比大场面开局更有代入感
- 社会热点提供故事基调，但不直接照搬

## 角色身份

引导作者给主角一个"不平的起点"——读者共情的不是完美，是困境。

> **提问：** "你的主角开局最糟糕的地方在哪里？他/她失去了什么？"
> **案例：** 主角剩三个月寿命→破罐破摔疯狂花钱→意外中奖→现在不想死了。目标是活的，被剧情推着变。

| 类型 | 例子 |
|------|------|
| 生存绝境 | 只剩三个月寿命、被困绝地 |
| 身份绝境 | 被灭门、被诬陷命案在身 |
| 债务绝境 | 欠了几个亿、被迫替人还债 |
| 关系绝境 | 被最信任的人背叛、被全家抛弃 |

- 不给完美开局——天崩开局更好写，逆风翻盘读者才想看
- 不给固定目标——目标是活的，剧情发展会改变它

## 画面感

帮助作者把脑中的画面转化为可写的素材。

> **提问：** "你现在看到的是什么画面？"
> **逐层展开：** 视觉（颜色/光线/物体）→ 听觉（什么声音/远近/节奏）→ 嗅觉（什么味道）→ 触觉（温度/质感/痛感）→ 内心（角色此刻在想什么）

- 写作先有画面，才有文字
- 如果作者卡住了，让他用手机拍一段类似的场景再回来写

## 写作风格

引导作者找到自己的"那道菜"。

> **提问：** "你最喜欢的三本小说/三部电影是什么？它们哪里最吸引你？"
> **核心：** 十个人炒西红柿炒鸡蛋，味道各不相同——你的阅历、你对事情的看法，才是别人学不走的东西。

- 不盲目追榜——热点会过时，但你的风格不会
- 建议作者先写完一本 30 万字小说。写完才能体会"掌控故事"是什么感觉
